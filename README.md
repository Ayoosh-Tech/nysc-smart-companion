NYSC Smart Companion
NYSC Smart Companion is a Progressive Web App (PWA) designed for Nigerian NYSC corp members. It provides tools to help manage your service year efficiently, including:
Task Management: Add, track, and delete daily activities.
Finance Tracker: Record income and expenses to monitor your balance.
Service Progress: Calculate POP (Passing Out Parade) countdown and track days served.
CDS & PPA Reminders: Keep up with essential NYSC activities.
Dark Mode Support: Easy on the eyes for day and night usage.
This app is lightweight, works offline, and can be installed as a mobile app for quick access.
Note: This app is not officially affiliated with NYSC; it is a personal productivity tool for corp members.
